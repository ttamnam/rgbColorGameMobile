Welcome to the RGB Color Game!

How to play:

  1. RGB color values represent the intensity of red, green, and blue in a color.
  2. An RGB value is usually written as rgb(red, green, blue), where each component is a number from 0 to 255.
  3. Your task is to select the color that matches the RGB value displayed.

Example: RGB(10, 200, 10)
